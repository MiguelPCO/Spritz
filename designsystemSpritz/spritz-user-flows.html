Flow 1: recomendación diaria Core loop
El motivo principal por el que el usuario abre la app cada mañana
Abrir app
Entry point diario
→
AI analiza contexto
Clima + hora + historial + colección
→
Today's Pick
Recomendación + razón + foto frasco
→
Usar hoy
1 tap para registrar
→
Confirmación
Animación spray + log guardado
Desde Today's Pick: [Otra opción] → Nueva recomendación (mismos factores)
Desde Today's Pick: [Cambiar ocasión] → Selector (trabajo, cita, casual, formal) → Nueva rec
Desde Today's Pick: [Tap en fragancia] → Fragrance Detail screen
Flow 2: explorar colección Daily use
Navegar y gestionar las fragancias que ya tiene el usuario
Tab Wardrobe
Bottom nav
→
Grid colección
Cards compact en 2 cols + search + filtros
→
Fragrance detail
Info completa + notas + historial de uso
→
Usar hoy
Mismo CTA que en Today's Pick
Desde Grid: [Filtrar] → Tags de familia olfativa (horizontal scroll)
Desde Grid: [Buscar] → Search con resultados en tiempo real
Desde Grid: [Toggle vista] → Cambiar entre grid y lista
Desde Detail: [Editar] → Edit mode (cambiar tags, notas, foto)
Desde Detail: [Eliminar] → Confirmación → Volver a grid
Flow 3: añadir fragancia Onboarding critical
Cuantas más fragancias tenga el usuario, mejores las recomendaciones
Botón [+]
Centro del bottom nav
→
Search DB
Buscar por nombre o marca
→
Seleccionar resultado
Pre-fill: nombre, marca, notas, foto
→
Personalizar
Tags ocasión, foto propia, notas
→
Guardado
Animación spray + aparece en wardrobe
Si no encuentra en DB: [Añadir manual] → Form completo (nombre, marca, foto, familia, tags)
Quick add desde Discover: [+ Add] → Salta al paso Personalizar directamente
Flow 4: descubrir nuevas Growth
Expandir la colección con fragancias que complementen lo que ya tiene
Tab Discover
Bottom nav
→
AI suggestions
"Basado en tu colección"
→
Fragrance detail
Info + por qué te podría gustar
→
Add to wishlist
Guarda para futuro
Secciones en Discover: Para ti (AI) / Trending / Por estación / Nuevos lanzamientos
Desde detail: [Comprar] → Link externo a tienda
Flow 5: historial de uso Retention
Ver patrones, racha de uso, y fragancias olvidadas
Profile / Log
Acceso desde profile
→
Calendar view
Mes con dots de color por familia
→
Day detail
Qué usaste + condiciones del día
Stats visibles: Más usada / Menos usada / Racha actual / Familias favoritas %
Nudge: "No has usado X en 30 días" → integra en recomendación AI
